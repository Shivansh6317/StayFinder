# 🏨 StayFinder

A Hotel Listing Application built using React and Fetch API.

## Features

- Hotel Listing
- Search Hotels
- Filter by Location
- Sort by Price / Rating / Name
- Pagination
- Hotel Details
- Wishlist
- Local Storage
- Responsive Design

## Tech Stack

- React
- React Router DOM
- Fetch API
- Context API
- CSS

## Installation

```bash
npm install
npm run dev
```
